# WORK ORDER 0004 — NAMENSKONVENTION & ZIELSTRUKTUR

## Goal
Lege **eine endgültige Namens‑ & Repo‑Konvention** fest, die business‑tauglich ist.

## Scope
Alle Sheratan‑Repos

## Task
1. Definiere eine verbindliche Namensstruktur, z. B.:
   - `sheratan-core-*`
   - `sheratan-tool-*`
   - `sheratan-exp-*`
   - `sheratan-product-*`
2. Ordne jedem existierenden Repo:
   - einen **Zielnamen**
   - einen **Migrationsstatus** zu

## Definition of Done (DoD)
- Datei `NAMING_CONVENTION.md`
- Tabelle:
  - aktueller Name
  - Zielname
  - Aktion (`keep | rename | archive`)

## Constraints
- Noch **keine** Umbenennungen durchführen
- Nur Plan

## Deliverable
